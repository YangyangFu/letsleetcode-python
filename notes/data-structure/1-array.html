Sliding Window#

Sliding window is a type of left and right pointer. This technique is very efficient to solve substring or subsequence problem.

left, right = 0, 0
while right < len(s):
    # enlarge window
    window.add(s[right])
    right += 1

    # shrink window
    while (window needs shrink):
        window.remove(s[left])
        left -= 1
